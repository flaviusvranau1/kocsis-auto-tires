# Kocsis Auto Tires — Centrul de Anvelope Bistrița

Landing page demo de tip *3D scroll cinematic* pentru un centru de anvelope din Bistrița.
Construit cu **Three.js** (anvelopă 3D randată în timp real, controlată de scroll) și **GSAP ScrollTrigger**.

- Single-file (`index.html`), fără build — se poate găzdui oriunde.
- 100% responsive, în limba română.
- Secțiuni: Hero 3D · Servicii · Branduri · Proces · Despre · Recenzii · Contact.

**Demo live:** publicat prin GitHub Pages.

> Demo de prezentare. Datele de contact, programul și recenziile sunt exemple și pot fi înlocuite cu cele reale.
